### number

在js中一个变量可以设置任意的数值

```
var num = 0;
num = 's';

在ts中需要设置的是
var num : number = 3;
```

没有对int和float进行一个明显的区分，都是number类型

```ty
var num : number = 4;
num = 3.4F
```
