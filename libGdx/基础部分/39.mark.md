```
private Palette load(AssetManager assets, String fontName) {
  AssetDescriptor<Palette> descriptor = getDescriptor(fontName);
  assets.load(descriptor);
  assets.finishLoadingAsset(descriptor);
  return assets.get(descriptor);
}
```
