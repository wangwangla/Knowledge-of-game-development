# 动画

Action是的sprite动起来，动画在每一帧根据一定的时间线和游戏角色的形状修改位置或者属性。一帧中存在多个物体在执行动画，所有动画需要管理起来。

优先级：在动画过程中，每一帧最先执行的优先级最低。会被后来的修改.

## 曲线

几乎所有的游戏动画系统都基于插值完成的，两个端点，一个时间

### 标量插值

标量插值的计算方式

```java 
float lerp(float v0,float v1,float t){
    return (1-t)*v0 + t * v1;
}
```

### 矢量插值

一个只变为两个只计算插值

### 矩阵插值

无论什么插值都可以看作是一个插值的组合。
