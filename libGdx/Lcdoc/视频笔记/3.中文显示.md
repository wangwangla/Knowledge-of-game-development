BitmapFont类介绍

- 是一个处理刚才创建好的两个文件，使用SpriteBatch绘制的
- 用途：用于设置静态文件，也可以设置颜色大下等

代码

```java
spriteBatch = new SpriteBatch();
font = new BitmapFont(Gdx.files.internal("Demo.fnt"),Gdx.files.internal("Demo.png"),false);

//绘制
spriteBatch.begin();
font.draw(spriteBatch, "见贤思齐", 30, 30);
spriteBatch.end();
```
